1. Platform summary

QS Assets is a unified enterprise platform for IT and non-IT asset management, discovery, network monitoring, patch and vulnerability management, ITSM, CMDB, fleet/GPS, CCTV, VDI, procurement, automation, reporting, and compliance — delivered as multi-tenant SaaS or self-hosted Docker deployment.

Lifecycle model: Procure → Discover → Inventory → Monitor → Secure / Patch → Service → Retire.

19. Known gaps (honest summary)

The following are commonly marketed in enterprise RFPs but are not yet complete in QS Assets as of July 2026:

  • Full MFA and SAML/OIDC enterprise hardening with acceptance tests green
  • SSDLC change gates, CSAT, and IMAP ticket ingest
  • Fleet geofencing, CCTV video wall, VDI session metric charts
  • NAC RADIUS CoA live quarantine on switches
  • Scheduled report email delivery and eight role-specific dashboards
  • Postgres RLS bypass hardening for all tenant paths

Explicit non-goals (current wave): Qualys-class proprietary vuln signatures; ServiceNow IntegrationHub-style visual flow designer; Kafka/NATS as primary event bus; remote super-admin access into customer on-prem databases.
